#### Run to help

```bash
gendiff - h
```
```bash
usage: gendiff [-h] [-f {stylish,json,plain}] first_file second_file

Generate diff

positional arguments:
  first_file
  second_file

optional arguments:
  -h, --help            show this help message and exit
  -f {stylish,json,plain}, --format {stylish,json,plain}
                        set format of output
```
